HTML Block Elements vs Inline Elements

Block Elements：總是在新的一行開始高度，元素寬度預設會撐到最大，使其占滿整個容器。
ex. <div>、<p>......

Inline Elements：和其他元素都在同一行上，元素的寬高由它的內容撐開。
ex. <span>、<a>......
